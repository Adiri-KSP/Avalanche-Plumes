NOTES-

Tweakscale appears to be incompatible with SmokeScreen, this is not something I can fix, so expect the smoke effects to be misaligned when tweakscaling boosters.

If your computer is weak and/or you are getting FPS drops with large amounts of boosters, go into Avalanche/Templates/SmokeScreen, open the SS_LimiterPatch, and change the number from 20,000 to a lower number like 5,000. If that does not solve the FPS issues please make a bug report.

 =

Requires: Module Manager, Waterfall, Smokescreen

Currently incompatible with: RealPlume, RSMP/SWE

 =

Installation Instructions

-Remove any versions of SWE/RSMP/Avalanche

-Download the most recent release

-Unzip into GameData

-Enjoy!
